Given a random variable X over space R, corresponding probability function f(x) and "value function" u(x), the expected value of u(x) is given by \begin{equation*}E = E[u(X)] = \sum_{x \in R} u(x) f(x)\end{equation*} provided X is discrete, or \begin{equation*}E = E[u(X)] = \int_R u(x)f(x) dx\end{equation*} provided X is continuous.
